# Logo del OSAC Knowledge Bot

Por favor, coloca el logo del robot en esta carpeta (`frontend/public/`) con el nombre:

**`logo.png`**

O si prefieres usar otro formato:
- `logo.jpg`
- `logo.svg`

El logo se mostrará automáticamente en:
- La barra de navegación superior
- Las páginas de Login y Registro

Asegúrate de que el archivo tenga un tamaño razonable (recomendado: máximo 500x500px para PNG/JPG, o formato SVG para mejor calidad).
